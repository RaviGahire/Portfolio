# Color Schema

## Base Background
#16171d — Main dark fa7444background

# Text Colors

#ffffff — Primary text on dark backgrounds
#867e8e — Muted / secondary text
#3b3440 — Subtle borders and dividers

# Purple Accent Hierarchy

#6c3bff — Electric purple (primary actions, buttons, links, highlights)
#b39aff — Soft lavender (secondary purple, hover states, accents)
#140033 — Deep wine purple (subtle dark backgrounds, code blocks)

# Gradient Signature
Purple-to-orange glow gradient used sparingly for primary buttons, CTAs, and banners:
Core: #6c3bff → #ba189a → #f4563a → #fa7444
Radial accents: #a200ff and #

# Secondary Accents (optional)
#32f3e9 — Cyan/teal
#22ff73 — Bright green

# Card Surfaces

Slightly lighter than base background for elevation (using #1e1f27 or similar soft dark tones)
Borders

#3b3440 (nickel) — Subtle dividers in dark mode
#e5e4e7 — Light borders (if used in light mode sections)
